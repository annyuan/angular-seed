  • a tags

    a tags come just with their text underline removed. Browser color defaults will apply. This is to minimize overrides, as a tags are more often than not styled as buttons.

  • =circled-ol

    Gives the ordered list numbers a circular colored background.

      Optional arguments:
    • $padding-left: ($line-height * 1.25) + 'em'

      The padding on the list item where the circled number will sit.

    • $left: -.35em

      The left position of the circled number.

    • $top: 0

      The top position of the circled number.

    • $font-size: inherit

      The font size of the circled number.

    • $line-height: 2

      The line height of the circled number.

    • $c: white

      The color of the circled number.

    • $bgc: $c-light-gray

      The background color of the circled number.

    1. First ordered list item
    2. Second ordered list item
    3. Third ordered list item
              
ol
  +circled-ol

  • =bulleted-list

    Places a bullet (a symbol set "record" icon) before each list item.

      Optional arguments:
    • $padding-left: 1.5em

      The padding on the list item where the bullet will sit.

    • $top: .15em

      The top position of the bullet.

    • $left: .5em

      The left position of the bullet.

    • $font-size: .35em

      The font size of the bullet.

    • $color: $c-light-gray

      The color of the bullet.

    • First bulleted list item.
    • Second bulleted list item.
    • Third bulleted list item.
