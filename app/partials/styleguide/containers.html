Containers

  • =panel

    Wraps stuff in a colored background.

      Optional arguments:
    • $bgc: false

      The background color of the panel.

    • $lines-vertical-padding: false

      (integer) The vertical padding of the panel in terms of multiples of $base-line-height.

    • $lines-horizontal-padding: false

      (integer) The horizontal padding of the panel in terms of multiples of $base-line-height.

    • $font-size: $base-font-size

      The font size is used to generate padding values in em's that respect vertical rhythm.

    I am sitting in a blue panel.

              
p
  +panel($bgc: lighten($c-blue, 50%), $lines-vertical-padding: 2)

  • vertical rhythm by Compass

    We use the CSS border property to create dividers. Compass provides mixins for creating borders that respect vertical rhythm - they are documented here.

    I have a bottom border but vertical rhythm on this page is maintained.

              
p
  border-bottom-color: $c-blue
  +trailing-border($font-size: $fs-p, $width: 4px, $lines: 4)
