Buttons

  • button tags and the button class

    Since there is a button tag, the button class deserves to be in the markup. They come with minimal styles, but if they are being included within a frequently used component (such as a form), then they should hopefully have additional style settings pre-applied.

    And me 
              
%button Click me
%a.button And me

  • Disabled buttons

              
%button.disabled Disabled

  • @mixin button-colors(...)

      Required arguments:
    • $c

      Text color

    • $bgc

      Background color

    • $bgc-hover

      Background color on hover

      Optional arguments:
    • $c-hover: $c

      Text color on hover

              
%button I'm orange

button
  @include button-colors(white, $c-orange, lighten($c-orange, 10%))

  • @mixin button-sizes(...)

      Required parameters
    • $font-size
    • $line-height
    • $lines-horizontal-padding
              
%button Big button

button
  @include button-sizes(20px, 12, 7)

  • %button-green

    Creates a green button.

    @include button-colors(white, $c-green, lighten($c-green, 10%))

    This is a prepackaged setting for buttons that is included here because it will often be used.

              
%button I'm green

button
  @extend %button-green

  • %button-medium

    Creates a medium-sized button.

    @include button-sizes(16px, 9, 5)

    This is a prepackaged setting for buttons that is included here because it will often be used.

              
%button I'm medium-sized

button
  @extend %button-medium
